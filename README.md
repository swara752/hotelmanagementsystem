# Hotel Management System

A simple **Hotel Management System** implemented in **Java** that lets you manage basic hotel operations such as checking in guests, checking out guests, and viewing available rooms.

This project is ideal for learning object-oriented programming and building console-based Java applications.

---

## 🏨 Features

✔ Display available rooms  
✔ Check-in a guest to a room  
✔ Check-out a guest from a room  
✔ Simple text-based user interface

---

## 🛠️ Technologies Used

- Java (Core)  
- Scanner for input

## 🚀 Getting Started

### Requirements

Make sure you have the following installed on your system:

- Java JDK (version 8 or above)  
- Any Java-compatible editor or IDE (e.g., IntelliJ, Eclipse, VS Code)

---

### 📌 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/swara752/hotelmanagementsystem.git
   cd hotelmanagementsystem

Compile the Java code

javac HotelManagementSystem.java

Run the program

java HotelManagementSystem
📚 How It Works

When you run the program, it displays a simple menu:
